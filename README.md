# 🚀 AI Student Retention Intelligence Dashboard

An AI-powered dashboard that predicts student drop-off risk and provides actionable insights to improve retention in edtech platforms.

---

## 📌 Live Demo
👉 https://snehaaggarwal24-student-retention-dashboard-app-bg9lku.streamlit.app/

---

## 🎯 Problem Statement

Many students sign up for free trials on edtech platforms but drop off before converting. This leads to reduced engagement and business loss.

This project predicts which students are likely to drop off and helps take proactive actions.

---

## 💡 Solution

This dashboard uses machine learning to:
- Predict student drop-off risk
- Provide probability-based insights
- Explain key factors behind predictions
- Suggest actions to improve retention

---

## ⚙️ Features

- 🔮 Drop-off Prediction using ML (Random Forest)
- 📊 Dynamic Probability Score
- 🧠 Explainable AI (Key Drivers of Drop-off)
- 📈 Interactive Data Visualizations
- 🎯 Personalized Insights
- 💡 Smart Recommendations for retention
- 🌐 Fully deployed web app

---

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit

---

## 📊 How It Works

1. User inputs student details (sessions, quiz score, activity, etc.)
2. Model predicts drop-off probability
3. Dashboard shows:
   - Risk level (High/Medium/Low)
   - Personalized insights
   - Key influencing factors
   - Suggested actions

---

## 🧠 Example Insights

- Low sessions → High drop-off risk  
- Low quiz score → Learning difficulty  
- High inactivity → Likely churn
